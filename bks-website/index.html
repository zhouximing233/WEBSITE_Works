游戏规则

贝壳杀基本规则

首先每人开局有对应血量+2能量（如果无技能更改）然后回合开始，秘密的选取行动并同时揭晓，行动有10种：

1. 普攻

消耗1能量，造成1点攻击伤害

2. 重击

消耗2能量，造成2点攻击伤害

3. 铁山靠

消耗3能量，造成3点攻击伤害

4. 防守

消耗1能量，进行1点防御

5. 驻守

消耗2能量，进行2点防御

6. 闪避

消耗3能量，进行3点防御

7. 自刎

消耗0能量，自己-1血量

8. 能量回复

回复1点能量

9. 血量回复

消耗1能量，回复1点血量

10. 斗转星移

消耗3能量，选取一个目标，其本回合的行动视为你对其使用

行动结算示例

例如：我使用"3.铁山靠"（-3能），而你使用"5.驻守"（-2能），那么结算为：我-3能量，你-2能量并-1血量。

回合行动规则：若是在你的回合内，你可以选择任意行动；若不在你的回合内，你只能选择除攻击行动（1-3）以外的行动。

回合流程：行动结算 → 主动技发动（若无阶段要求） → 主动技结算 → 结束阶段 → 回合结束。

斗地主模式

游戏规则

农民：无额外效果，在地主回合内有2个行动，但在该回合内最多选择一次回能，且防御必须点明防御第几个行动。

例如：地主第1行动打2，农民第2行动防2，则视为地主打农民2点伤害。

地主：在其回合内行动开始前+1能，且初始血量上限及血量+1，可在其回合内选取两个行动（分先后）。

例如：地主2能，可第一行动先回能，第二行动打3点。

角色设计要求

  • 必须是学生
  • 需要提交设计角色的清晰照片
  • 强度顺序：标 ＜ 界 ＜ 谋 ＜ 神
  • 设计的角色需符合角色定位和要求

人物技能关联特殊模式

博弈

描述：双方可选择红花色/黑花色

都红：双方各+1血

一红一黑：黑+1血+1能，红无效果

都黑：双方各受到1点伤害

此为角色技能关联的特殊模式，需要特定角色技能触发

决斗

描述：技能全部无效，若无决斗内技能提示

初始2能量，血量=回合外的血量（无盾）

连续进行符合规则的八个回合

无特殊提示则输的人-1血

同归于尽则算发起者输

此为角色技能关联的特殊模式，需要特定角色技能触发

数劫

描述：发起者可以选择1-5的一个数字，若对方也选择了相同数字，则视为发起者对其造成一点伤害（若无技能描述则为一点）。

此为角色技能关联的特殊模式，需要特定角色技能触发
